# Security Policy

If you discover a security issue, do NOT open a public issue.

Please report it privately:

contact@extraktr.com

We will acknowledge receipt within 48 hours and work on a fix.

## Scope

This applies to:
- CLI
- API usage
- Public-facing integrations

Please include:

• Description of the issue
• Steps to reproduce
• Impact assessment if known

## Notes

This repository does NOT contain the full backend or extraction engine.
