# Extraktr Workflow

Extraktr converts conversations into execution.

Core flow:

1. Input
- Slack thread
- Gmail thread
- Raw text
- CLI input

2. Extract
- Tasks
- Decisions
- Risks
- Summary

3. Review
- Edit tasks
- Confirm accuracy
- Adjust formatting

4. Send
- ClickUp
- Jira
- API destination

Outcome:
Work moves from conversation → system without manual effort.
